# qa_python
 Реализованные тесты

### 1. Проверка добавления новой книги
- **Цель**: убедиться, что новая книга успешно добавляется в систему.
- **Описание**: тест добавляет новую книгу и проверяет её наличие в списке книг.

### 2. Проверка установки жанра книги
- **Цель**: проверить, что жанр успешно устанавливается для существующей книги.
- **Описание**: тест добавляет книгу, устанавливает для неё жанр и проверяет, что жанр установлен корректно.

### 3. Проверка получения списка книг с определённым жанром
- **Цель**: убедиться, что система корректно возвращает список книг с заданным жанром.
- **Описание**: тест добавляет несколько книг с разными жанрами и проверяет, что список книг с выбранным жанром возвращается корректно.

### 4. Проверка списка книг для детей
- **Цель**: проверить, что система корректно определяет книги, подходящие для детей.
- **Описание**: тест добавляет книги с разными жанрами и проверяет, что в списке книг для детей присутствуют только те книги, которые не имеют возрастного рейтинга.

### 5. Проверка добавления книги в избранное
- **Цель**: убедиться, что книга успешно добавляется в список избранного.
- **Описание**: тест добавляет книгу в систему, затем добавляет её в избранное и проверяет наличие книги в списке избранного.

### 6. Проверка удаления книги из избранного
- **Цель**: проверить, что книга корректно удаляется из списка избранного.
- **Описание**: тест сначала добавляет книгу в избранное, затем удаляет её и проверяет, что книга больше не присутствует в списке.

### 7. Проверка пустого списка избранного
- **Цель**: убедиться, что при отсутствии книг в избранном система возвращает пустой список.
- **Описание**: тест проверяет, что метод `get_list_of_favorites_books` возвращает пустой список, если ни одна книга не добавлена в избранное.

### Дополнительные тесты (проверка граничных условий)

#### Тест добавления книги с пустым названием
- **Цель**: проверить обработку случая, когда название книги — пустая строка.
- **Описание**: тест пытается добавить книгу с пустым названием и проверяет, что она не появляется в системе.

#### Тест добавления книги со слишком длинным названием
- **Цель**: проверить ограничение на длину названия книги.
- **Описание**: тест создаёт название книги длиннее допустимого лимита и проверяет, что такая книга не добавляется в систему.
